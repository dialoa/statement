Cross-references with the statement filter

Julien Dutant

Abstract
Demonstrates cross-references with the statement filter. Filler text is used to make sure the links point to the theorem rather than a section heading. Theorems are counted within section to demonstrate numbered crosslabels.

Formatting tests:

  • @mytheorem, 1.1
  • [@mytheorem], (1.1)
  • [-@mytheorem], (1.1)
  • [See @mytheorem below], (See 1.1 below)
  • [See @mytheorem below; and @the-old-theorem as well], (See 1.1 below; The Old Theorem as well)
  • []{#mytheorem}, 1.1
  • [Theorem <>]{#mytheorem}, Theorem 1.1

Bug test. If a theorem’s ID is a duplicate of another element’s, make sure the filter doesn’t crash when we try to reference it with 1.2 or 1.2. The filter will warn, change the theorem id and process references to it the best it can.
